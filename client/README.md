# Playlist — client (React + Vite)

This is the **frontend** of the Playlist app. It is a React app built with Vite.

👉 For everything — how to run the project, the API, the folder guide, and the
documentation — see the **main [README](../README.md)** in the project root.

```bash
npm install
npm run dev      # start the dev server (needs the API running on :3000)
```
